疫情期间发现一些「美好」

2020-04-08
E.能量
约 1184 字 预计阅读 3 分钟 | 阅读

文章目录

每个人对「美好」的定义是不同的，特别是在这个特殊时期，似乎无论如何都谈不上「美好」这个词，前线的医务人员在奋战，普通百姓在居家隔离，病者痛苦，医者忙碌。

然而生活还是要继续的，特殊时期更需要找到特殊的「美好」来让生活更有意义，否则只能在家宅着窝着长肉，心情着实会太糟。

以下是我在这三个月间「寻到」的「美好」，靠着他们让我这段时间以来每天都在忙碌着，无比的充实。

  1. Notion，一个 all-in-one 的网上协作空间，可个人或团队使用，以 block 为单位组织内容，支持文本、Markdown 语法、database、media embeds等，可以发掘出很多玩法，很有意思。
  2. 子弹笔记，Bullet Journal，一种由特定结构和符号规则构成的笔记方法，用来做个人管理，非常新颖并且有效率。目前我每天都在用这个方法做记录和跟踪。
  3. 结合 1 和 2 我开发了 Notion 版的子弹笔记，链接在这里。
  4. Coursera 上的机器学习课程 。这门课是斯坦福于2011年上线的三门在线课程之一，由 Coursera 的 Cofounder 吴恩达主讲，Coursera就不说了，MOOC始祖，在线教育领域最成功的网站之一。这门课让我感受更多的是中外教学模式的差异，尤其是在高等教育中教师与学生之间的关系、教师的授课风格等，真心觉得创新教育方面，我们落后的太多太多，不是近年来喊些口号做些改革就能改变得了的，更多的是深层次的文化内涵、社会价值文化等方面的落后。
  5. Github，很早以前就知道它，但由于我不是程序员，所以从来也没用过它，最近想学习一些程序方面的课程，锻炼一下中老年的大脑，于是玩起了Github，体会到的乐趣有：
    1. Github Pages，于是通过 Hugo 建立了这个网站。
    2. Hugo，轻量级的静态页面发布系统，markdown 写作，命令行发布，上手简单。
    3. Youtube-dl，免费的命令行油管视频下载，可配置代理，好用。
    4. Clementine，Mac 下的本地音乐播放器，功能丰富，够用。实在不喜欢自带的「音乐.app」。
  6. 读书，这几个月在家时间长，有充足的时间消化库存，目前保持每周一本的速度，其中比较有感触的是《延展：释放有限资源的无限潜能》，翻看京东的订单记录才发现，这本书居然2018年我就买了，放在书架里近 2 年了😭，果然「买书如山倒，读书如抽丝」。结合近年心境，越发觉得本书所提倡的观点心有戚戚，也许是岁数大了，缺少一些与世争锋的锐气，更多的是一种沉下心来静心观己的平和，「延展思维」就是充分发挥和挖掘已有资源的价值，寻求更积极和美好的结果。
  7. Heart Sutra × Ikkyu-ji Temple,Kyoto,Japan - 薬師寺 寛邦 キッサコ，这是前两天看到了，我认为是目前听到的最好听的「心经」。通过YouTube-dl下载了mp3，正在每日循环播放。相当惊艳。视频地址在这里。

有句话说，生活的全部意义在于无穷地探索尚未知道的东西，在于不断地增加更多的知识。在我看来，「探索」是一件能够收获「美好」的事，希望在未来的岁月中，我依然能保持这种对「探索」的热爱。
